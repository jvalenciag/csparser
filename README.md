# Copylight 2015 

## Dependencias:
    sudo apt-get install python-bs4

## Ejecución:
    python csparser.py

## Directorios:
* data/
    - Archivos en formato csv por carrera y proceso, uno para Postulantes-Vacantes y otro para Puntaje Máximo y Mínimo.
* cache/ 
    - Archivos html descargados por el parser
